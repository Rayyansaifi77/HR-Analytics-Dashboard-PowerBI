📊 HR Analytics Dashboard

## Project Overview

The HR Analytics Dashboard using Power BI is designed to help HR managers analyze employee attrition patterns and workforce data. 
This dashboard provides insights into employee demographics, attrition trends, job satisfaction, and department-level employee distribution.

The project helps organizations make data-driven decisions related to employee retention, recruitment strategies, and workforce planning.

## Technologies Used
- Microsoft Power BI
- Advanced Excel
- Data Visualization
- Statistics

## Key Performance Indicators (KPIs)
- Employee Count
- Attrition Count
- Attrition Rate
- Active Employees
- Average Employee Age

## Dashboard Visualizations
- Attrition by Gender (Lollipop Chart)
- Department-wise Attrition (Pie Chart)
- Employees by Age Group (Bar Chart)
- Job Satisfaction Rating (Square Chart)
- Education Field-wise Attrition (Bar Chart & Donut Chart)

## Outcome
This dashboard provides actionable insights into employee attrition and workforce structure, helping HR teams identify problem areas and improve employee retention strategies.
